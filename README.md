# Sites Monitor

### How to use:

- Clone repo

```shell
  $ git clone git@github.com:guilhermecaldeira2/go-site-monitor.git
```

```shell
  $ cd go-site-monitor.git
```

- Download Golang

- Execute:

```shell
  $ go run siteMonitor.go
```

- if want to modify the sites include in Sites.txt file
